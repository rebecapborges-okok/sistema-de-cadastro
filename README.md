Sistema de cadastro dos clientes de uma corretora de imóveis, com informações de contato, descrição das casas, fazendas ou terrenos disponíveis à venda ou parâmetros desejados para compra.
Este projeto foi feito para facilitar a organização de uma corretora de imóveis autônoma. Através do sistema de cadastro de clientes, a corretora terá informações sobre seus clientes de forma mais rápida e fácil do que numa agenda física ou WhatsApp. O sistema será responsável por armazenar dados de contato, descrição do imóvel à venda e informações sobre as qualidades desejadas para compra.
